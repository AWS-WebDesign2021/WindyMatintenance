About This Webpage

This was the first real web page I ever made, it includes the simple default fonts and formatting. There was no CSS styles used or Java Script. The assignment asked us to create a small web page using lists and different text types that gives us a brief introduction. We were also asked to use photos and YouTube videos. Looking back I believe I did quite well to create an organized simple page using the elements I knew at the time. If I did this web page now it would likely involve colors and sections and maybe a navigation bar, but this page does its job quite well. I could have added comments while creating it so that the code was more organized.
